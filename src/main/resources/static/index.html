LPS - Prototype v1

Selbst erscheinen

Um selbst auf der Karte zu erscheinen, muss eine App installiert werden, um Bluetooth Low Energy Pakete zu verschicken. Dazu eignet sich auf Android die App nRF Connect aus dem Playstore.

Auf iOS erlaubt diese App keine Erstellung eigener Services und kann deswegen nicht benutzt werden.

Mit den folgenden Schritten kann ein Advertiser erstellt werden, der LPS-konform ist.

  1. Advertiser erstellen
  2. Manufacturer Data hinzufügen. Company ID muss hier 0x504C sein.
    Das Datenfeld hat folgendes Muster: 0x53XXXX23.
    Die 'X' sind durch eine im Raum eindeutige 16-Bit ID zu ersetzen.

    Ein Beispiel für ein korrektes Datenfeld ist: 0x53000123

  3. Options ausklappen und Tx Power Level auf 1 setzen. Dadurch wird die Ergebnisqualität verbessert.
  4. Service starten. Der Service sollte nun vom System aufgenommen werden und auf der Karte erscheinen.
